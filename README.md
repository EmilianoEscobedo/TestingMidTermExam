## Segundo Parcial - Pruebas de Software - ISTEA

Se tiene la clase <ColaPrioridad.cs>, se requiere que escriban e implementen casos
de prueba unitarios que permitan probar todos los métodos de la misma. Además,
los test deben dar una cobertura del al menos el 70%.
Esta clase se compone de cuatro funciones:
- Encolar: agrega un elemento a la cola
- Desencolar: quita el elemento de mayor prioridad de la cola
- getMayorPrioridad: lee cuál es el valor del elemento de mayor prioridad
- IsEmpty: valida si la cola está vacía
